# AGENTS.md

## Repository Purpose

This repository exists to extract knowledge from LLMs.

We write articles about any subject using GPT-5.5, then save those articles as Markdown files in this repository.

## Working Instructions

- Always read this `AGENTS.md` file before starting any task in this repository.
- Preserve user-created work and do not overwrite unrelated changes.
- Prefer clear, structured Markdown for knowledge articles.
- Make articles useful as standalone references, with headings, context, and scope notes.
- When a subject is too large to cover exhaustively, be explicit about the limits of the article while still covering the major periods, concepts, people, or events.
- Keep filenames descriptive and readable.
- Use existing repository organization when adding new articles.

## Style

- Write in a direct, informative style.
- Favor accuracy and clarity over flourish.
- Use lists, timelines, tables, and glossaries when they make the article easier to use.
- Avoid unsourced claims that depend on fast-changing current events unless they have been verified.
